# schema-validator

Flexible data schema validation with custom rules engine

## Installation

```bash
pip install schema-validator
```

## Quick Start

```python
from schema_validator import Schema
client = Schema()
client.run()
```

## Features

- High performance and memory efficient
- Comprehensive error handling
- Well tested with full coverage
- MIT licensed
